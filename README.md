#Greenden

A responsive website for eco-friendly products built with HTML5 and Tailwind CSS.

Overview

Greenden is a static, fully responsive website showcasing sustainable products. It features a clean design that works seamlessly across all devices - mobile, tablet, and desktop. Built with pure HTML and Tailwind CSS, no JavaScript or dependencies required.

Features


Fully responsive design (mobile-first approach)
Semantic HTML5 markup
Smooth transitions and hover effects
Zero dependencies
Fast loading (static HTML)


Pages


Home - Landing page with introduction
Products - Product catalog with responsive grid
Contact - Contact form page


Tech Stack


HTML5
Tailwind CSS


Project Structure

greenden/
├── index.html
├── products.html
├── contact.html
├── images/
└── README.md

Responsive Breakpoints

The site is optimized for all screen sizes using Tailwind breakpoints:


Mobile: 0px - 640px
Tablet: 641px - 1024px
Desktop: 1025px+


Customization

Edit HTML files directly to:


Change colors (Tailwind color classes)
Update content and images
Modify spacing and layout
Add new pages
